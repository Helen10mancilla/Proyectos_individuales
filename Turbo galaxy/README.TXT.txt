🚀 Turbo Galaxy 🚀
-------------------------------

¡Bienvenido a Turbo Galaxy!

Un videojuego de carreras espaciales con una vibra tropical, donde deberás esquivar cocos, meteoritos y conquistar la galaxia. ¡Diversión garantizada!

🎮 Cómo jugar:
1. Asegúrate de tener instalado **Python 3.11 o superior**.
2. Instala la librería **pygame** si aún no la tienes:
   > pip install pygame
3. Ejecuta el archivo principal:
   > python main.py
4. Usa las flechas del teclado para mover tu nave y esquivar los obstáculos.
5. ¡Llega lo más lejos que puedas y supera tu propio récord!

⚙️ Requisitos:
- Python 3.11 o superior
- pygame (librería de Python para juegos)

📂 Estructura de carpetas:
- `main.py`: Archivo principal para ejecutar el juego.
- `assets/`: Imágenes, sonidos y recursos gráficos.
